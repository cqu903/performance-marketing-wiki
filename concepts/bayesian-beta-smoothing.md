# 贝叶斯 Beta 平滑

用 Beta 共轭先验融合「大盘历史经验 + 素材实时样本」，输出稳定无偏差的后验转化率，解决 UAC 冷启动与稀疏转化场景的数据失真。是全 UAC 策略（tCPI / tROAS / oCPC / 深度转化）的小样本通用修正工具。^[raw/articles/20260712-uac-quantitative-analysis-handbook.md]

## 解决的痛点
新素材、小曝光素材（10 秒有效播放 < 100）样本稀疏，原始 EVC 转化率严重失真：
- 0 转化 = 判死刑（实际可能是潜力素材未跑出）
- 1 单小样本 = 虚高爆款（误判放量）

导致素材误判、模型冷启动偏差。

## 模型原理
EVC 转化服从二项分布，搭配 Beta 共轭先验，后验期望平滑公式：

$\hat{p}_{bayes} = \frac{\alpha + k}{\alpha + \beta + n}$

- α、β：大盘 Beta 先验超参数（历史经验）
- k：单素材 EVC 转化数
- n：单素材 10 秒有效播放数

## 先验参数标准化计算步骤
1. 取账户同版位成熟素材（有效播放 ≥ 1000），统计大盘总播放 N、总 EVC 转化 K。
2. 计算大盘基准转化率：$p_0 = K/N$。
3. 设定先验强度 S（冷启动账户 S=50，成熟账户 S=100~200）。
4. 求解超参数：$\alpha = p_0 \cdot S$，$\beta = (1-p_0) \cdot S$。

⚠️ 关键规则：Shorts、In-Stream、Rewarded 版位必须单独拟合先验，不可通用。

## 业务示例对比
大盘基准 EVC 转化率 2%，α=2，β=98：
- 新素材 n=30, k=0：原始转化率 0% → 平滑转化率 1.54%（修正全盘否定偏差）
- 新素材 n=10, k=1：原始转化率 10% → 平滑转化率 2.73%（修正虚高爆款偏差）

## 平滑后素材评级标准
- 优质潜力：平滑 EVC 转化率 > 1.2 倍大盘基准 → 放量、复制创意、扩量人群
- 中等待优化：0.8~1.2 倍大盘基准 → 小幅迭代素材细节
- 低效淘汰：< 0.8 倍大盘基准 → 直接关停止损

## 通用化：全 UAC 小样本数据通用修正
该模型适配所有 UAC 优化目标，替代传统「原始转化率 / 成本单点判断」：
- **tCPI（安装优化）**：新计划安装量稀疏时原始 CPI 极端，平滑后得真实获客潜力，避免错杀冷启动计划。
- **tROAS（营收优化）**：付费/入金样本极少，原始 ROAS 无参考性；基于大盘同类素材历史 ROAS 先验修正小样本投产偏差。
- **深度转化优化（注册/留资/付费）**：深度行为天然稀疏，平滑后稳定输出真实转化预期。

通用规则：所有 UAC 策略中，转化样本量 < 100 时统一禁用原始指标，全部采用贝叶斯平滑；大样本数据自动收敛至真实值，不影响成熟计划判断。支持分版位、分品类、分地区自定义先验。

## 与相关性分析的联动（双模型工作流）
- 大样本素材（有效播放 ≥ 1000）：用 [[creative-correlation-analysis]] 挖掘最优特征规律。
- 小样本素材（有效播放 < 100）：跳过相关性，用贝叶斯平滑修正转化率，实现公平评级。
- 四维诊断：高留存+高平滑 EVC（顶级，加预算复刻）/ 高留存+低平滑 EVC（留人不转化，优化种草 CTA）/ 低留存+高平滑 EVC（人群精准但开头薄弱，优化前 3 秒）/ 低留存+低平滑 EVC（淘汰）。

对 UAC RankScore 竞价：贝叶斯平滑精准预估真实转化价值 E(Value)、转化率 p(Conversion)，让 Bid 出价更贴合真实流量价值，避免高估/低估，同时减少模型学习噪音。
